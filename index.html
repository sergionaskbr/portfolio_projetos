Projetos em Ciência de Dados

Desenvolvimento de um painel gerencial para negócio com o Streamlit

Nesse projeto, trabalhamos com programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, juntamente com ferramentas de desenvolvimento web como o GitHub e o Streamlit, que foram usadas para a criação de um painel gerencial contendo as métricas de uma empresa marketplace de delivery de comida da India.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud, disponibilizado através de um link web.

Dashboard de uma plataforma de restaurantes hospedado no Streamlit

Através do uso da linguagem de programação Python, foi criado o dashboard de uma plataforma que contém mais de 7 mil restaurantes em 15 países. Dividida em visões de negócio (países, cidades e restaurantes), o painel interativo mostra as principais dimensões dos dados, envolvendo por exemplo quantidade e localização dos restaurantes por região, avaliação média por restaurante e por tipo de culinária, dentre muitos outros.

As ferramentas utilizadas foram:

  • Python
  • JupyterLab, Visual Studio Code
  • Terminal
